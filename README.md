## 📋 Overview

<a href="#vectorworks">Vectorworks</a> serves the design professions where Mac has traditionally been the platform of choice — architecture, landscape architecture, interior design, and entertainment production — providing the professional design tools these disciplines require in a Mac-first application rather than a Windows application adapted for Mac.

The <a href="#vectorworks">architectural practice use case</a> spans from small residential studios to large commercial architecture firms — Vectorworks scales from a single-architect practice using it for complete project documentation to large firms with multiple designers collaborating on complex projects. The platform's depth covers the full project lifecycle from concept through construction documentation. The <a href="#vectorworks">landscape architecture use case</a> in Vectorworks Landmark uses site-specific tools — terrain modeling, plant database with growth characteristics, site utilities, and the documentation tools that landscape practice requires — in a platform that understands landscape design rather than adapting architectural tools to landscape use.

The <a href="#vectorworks">entertainment production design use case</a> in Vectorworks Spotlight covers the specialized documentation needs of theatre, live events, broadcast, and film — lighting plots, rigging plans, stage layouts, and the production-specific objects and data structures that entertainment production design requires. <a href="#vectorworks">Education and student licensing</a> makes Vectorworks accessible for architecture and design school curricula — students learning professional design software used in practice rather than academic-only tools that don't transfer to professional environments.</p>

<a href="#vectorworks">Cloud sharing and collaboration</a> through Vectorworks Cloud Services enables team collaboration on shared project files — multiple designers accessing current project data and publishing rendered views and presentations through the cloud without complex file server infrastructure. <a href="#vectorworks">Python and Vectorscript scripting</a> automates repetitive design tasks and extends Vectorworks with custom tools — the scripting capability that professional practices use to adapt Vectorworks to their specific documentation workflows and standards.

<a href="#vectorworks">Annual subscription and perpetual licensing</a> provide different access models — subscription for ongoing updates and cloud services, perpetual for practices that prefer software ownership without annual renewal. <a href="#vectorworks">Vectorworks Service Select</a> provides subscribers with cloud storage, mobile viewing, and priority support alongside the software updates.</p>

## 💬 FAQ

<details>
<summary>How does Vectorworks compare to Revit?</summary>

Both are professional BIM platforms. Revit dominates large commercial architecture practice; Vectorworks is stronger on Mac, has a gentler learning curve, and includes the Landmark and Spotlight disciplines. File exchange between them is through IFC.

</details>

<details>
<summary>Can Vectorworks be used for interior design?</summary>

Yes. Space planning, interior layout, furniture and fixture libraries, and interior documentation are well-supported in Vectorworks.

</details>

<details>
<summary>Does Vectorworks Spotlight integrate with lighting control software?</summary>

Vectorworks Spotlight includes connections to Lightwright for lighting paperwork. Check current documentation for other integration support.

</details>

<details>
<summary>What file formats does Vectorworks import and export?</summary>

DWG, DXF, IFC, PDF, image formats, and others. Check the current Vectorworks documentation for the complete format list.

</details>

<details>
<summary>Does Vectorworks support Python scripting?</summary>

Yes. Python and Vectorscript both enable automation and custom tool development within Vectorworks.

</details>
